# SillyTavern 图片自动生成插件 (Image Auto Gen Pro)

一个功能强大的 SillyTavern 插件，使用 LLM 自动分析对话内容并生成精美图片。

## 功能特性

- 🤖 **独立 LLM 调用** - 使用 OpenAI 兼容接口，不影响当前对话模型
- 🎨 **智能提示词生成** - 自动分析对话场景、角色、情绪，生成专业 SD 提示词
- 🌐 **多语言支持** - 英文提示词用于生图，中文描述作为注释
- 🔧 **高度可配置** - 多种触发模式、标签设置、插入方式
- 📝 **实时日志** - 完整的操作日志，方便调试

## 安装说明

1. 将本文件夹复制到 SillyTavern 的 `extensions/third-party/` 目录下
2. 重启 SillyTavern 或在扩展管理中点击 "Reload Extensions"
3. 在侧边栏扩展菜单中找到 "图片自动生成 Pro"

## 配置说明

### 1. LLM 配置

- **API 基础地址**: OpenAI 兼容接口地址（如 `https://api.openai.com/v1`）
- **API 密钥**: 你的 API Key
- **模型**: 推荐使用 GPT-4 或 Claude 3，效果更好
- **温度**: 控制提示词的创意程度，0.5-0.8 为宜
- **最大 Token**: 单次请求最大 token 数

### 2. 提示词编辑

- **系统提示词**: 自定义提示词生成逻辑
- **必填正面标签**: 自动添加到所有英文提示词开头（如 `masterpiece, best quality`）
- **必填负面标签**: 作为 imagine 命令的负面提示词
- **禁用词汇**: LLM 生成时避免使用的词

### 3. 通用设置

- **触发模式**:
  - 自动: 角色回复后自动触发
  - 关键词: 仅在用户消息包含关键词时触发
  - 手动: 只通过 "立即生成" 按钮触发
- **图片插入方式**: 插入当前消息或创建新消息
- **中文注释位置**: 图片标题、图片下方或两者都

## 使用示例

### 自动模式
1. 启用插件，选择自动模式
2. 正常与角色对话
3. 每次角色回复后会自动生成并插入图片

### 关键词模式
1. 选择关键词模式，设置触发词（如 "生成图片"）
2. 在对话中说"帮我生成一张图片"
3. 插件会自动处理

### 手动模式
1. 选择手动模式
2. 对话结束后点击"立即生成"按钮
3. 根据最后几条消息生成图片
